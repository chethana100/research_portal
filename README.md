# Financial Research Portal

A simple and efficient tool for extracting structured data from financial statements.

## Features
- **Upload**: Support for PDF and image formats (Annual Reports, P&L Sheets).
- **Extract**: Automatic section-based extraction using AI.
- **Export**: Download results as a formatted CSV for analysis.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables:
   Create a `.env.local` and add your key:
   ```
   chethana=your_api_key_here
   ```
3. Run the portal:
   ```bash
   npm run dev
   ```

Built with Next.js.
